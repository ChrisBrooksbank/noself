# Navigation & Search

## Overview

Browse the full catalog of 30 Buddhist concepts with search, filtering, and category navigation.

## User Stories

- As a practitioner, I want to browse all concepts so that I can choose what to study
- As a practitioner, I want to search by name or keyword so that I can quickly find a teaching
- As a practitioner, I want to filter by category so that I can study related teachings together

## Requirements

- [ ] Concept catalog/list view showing all 30 concepts with title, pali name, and brief description
- [ ] Text search that filters concepts by title, pali, sanskrit, and brief content
- [ ] Category filter (e.g., three-marks, four-noble-truths, meditation, etc.)
- [ ] Smooth navigation between catalog view and individual concept view
- [ ] Responsive layout that works on mobile and desktop
- [ ] Visual indicator for concepts the user has already read

## Acceptance Criteria

- [ ] All 30 concepts appear in the catalog
- [ ] Search narrows results as the user types
- [ ] Category filter shows only matching concepts
- [ ] Navigation between views is seamless (no full page reloads)
- [ ] Layout is usable on a 320px-wide screen

## Out of Scope

- Sorting options beyond category
- Pagination (30 items fits in a single list)
- Bookmarking or favorites
